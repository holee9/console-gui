Components

Reusable UI components with usage examples

Buttons

Primary Button

Used for primary actions and CTAs.

Secondary Button

Used for secondary actions and alternatives.

Danger Button

Used for destructive actions.

Icon Buttons

Compact buttons with icons only.

XAML Usage

XAML 
<!-- Primary Button -->
<Button Content="Save"
        Style="{StaticResource Button.Primary}"
        Command="{Binding SaveCommand}"/>

<!-- Secondary Button -->
<Button Content="Cancel"
        Style="{StaticResource Button.Secondary}"
        Command="{Binding CancelCommand}"/>

<!-- Danger Button -->
<Button Content="Delete"
        Style="{StaticResource Button.Danger}"
        Command="{Binding DeleteCommand}"/>

<!-- Icon Button -->
<Button Style="{StaticResource Button.Icon}"
        Command="{Binding AddCommand}">
    <TextBlock Text="+" FontSize="20"/>
</Button>

Input Fields

Text Input

Standard text input for user data entry.

Input with Validation

Input fields with validation states.

Search Input

Input field optimized for search.

XAML Usage

XAML 
<!-- Standard Text Input -->
<TextBox Text="{Binding PatientName, UpdateSourceTrigger=PropertyChanged}"
         Style="{StaticResource Input.Standard}"
         hw:TextField.HintText="Patient Name"/>

<!-- Read-only Input -->
<TextBox Text="{Binding PatientId}"
         IsReadOnly="True"
         Style="{StaticResource Input.ReadOnly}"/>

<!-- Input with Validation -->
<TextBox Text="{Binding PatientId, UpdateSourceTrigger=PropertyChanged}"
         Style="{StaticResource Input.Standard}"
         Validation.ErrorTemplate="{StaticResource ValidationErrorTemplate}">
    <Validation.Error>
        <ValidationErrorInfo>Invalid patient ID format</ValidationErrorInfo>
    </Validation.Error>
</TextBox>

XAML Usage

XAML 
<!-- Modal Dialog -->
<Window xmlns="https://github.com/MahApps/MahApps.Metro"
        Style="{StaticResource MetroWindow}">
    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="*"/>
            <RowDefinition Height="Auto"/>
        </Grid.RowDefinitions>

        <!-- Header -->
        <Border Grid.Row="0"
                Background="{StaticResource Neutral.Surface}"
                Padding="16">
            <TextBlock Text="Modal Title"
                       Style="{StaticResource TextStyle.H2}"/>
        </Border>

        <!-- Content -->
        <ContentControl Grid.Row="1"
                        Content="{Binding ModalContent}"/>

        <!-- Footer -->
        <Border Grid.Row="2"
                Background="{StaticResource Neutral.Surface}"
                Padding="16">
            <StackPanel Orientation="Horizontal"
                        HorizontalAlignment="Right">
                <Button Content="Cancel"
                        Style="{StaticResource Button.Secondary}"/>
                <Button Content="Confirm"
                        Style="{StaticResource Button.Primary}"/>
            </StackPanel>
        </Border>
    </Grid>
</Window>

Toast Notifications

Success Toast

✓ Patient registered successfully

Error Toast

⚠ Failed to save. Please try again.

Warning Toast

⚠ Connection unstable. Changes may not sync.

Info Toast

ⓘ New update available. Refresh to apply.

XAML Usage

XAML 
<!-- Toast Notification -->
<UserControl x:Class="HnVue.UI.Components.ToastNotification">
    <Border Style="{StaticResource Toast.Success}">
        <StackPanel Orientation="Horizontal" Spacing="12">
            <TextBlock Text="✓"
                       Style="{StaticResource TextStyle.H3}"/>
            <TextBlock Text="{Binding Message}"
                       Style="{StaticResource TextStyle.Body}"/>
        </StackPanel>
    </Border>
</UserControl>

<!-- Show from ViewModel -->
this.ShowDialog(new ToastViewModel {
    Type = ToastType.Success,
    Message = "Operation completed successfully"
});

XAML Usage

XAML 
<!-- Data Grid with Custom Styling -->
<DataGrid ItemsSource="{Binding WorklistItems}"
          AutoGenerateColumns="False"
          Style="{StaticResource DataGrid.Medical}"
          SelectedItem="{Binding SelectedItem}">
    <DataGrid.Columns>
        <DataGridTextColumn Header="Accession No"
                            Binding="{Binding AccessionNumber}"/>
        <DataGridTextColumn Header="Patient ID"
                            Binding="{Binding PatientId}"/>
        <DataGridTextColumn Header="Name"
                            Binding="{Binding PatientName}"/>
        <DataGridTextColumn Header="Exam Date"
                            Binding="{Binding ExamDate, StringFormat='yyyy-MM-dd'}"/>
        <DataGridTextColumn Header="Procedure"
                            Binding="{Binding ProcedureName}"/>
        <DataGridTemplateColumn Header="Status">
            <DataGridTemplateColumn.CellTemplate>
                <DataTemplate>
                    <Badge Content="{Binding Status}"
                           Status="{Binding StatusType}"/>
                </DataTemplate>
            </DataGridTemplateColumn.CellTemplate>
        </DataGridTemplateColumn>
    </DataGrid.Columns>
</DataGrid>
