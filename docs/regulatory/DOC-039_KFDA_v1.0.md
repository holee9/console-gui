## 5. 주의 사항

1. 식약처 심사 시 추가 자료 요청에 대비하여 **영문 문서 번역본** 준비 필요
2. **GMP 적합 인정** 확보 필수 (ISO 13485 기반)
3. 사이버보안 가이드라인 강화 추세 — **SBOM 및 취약점 관리 계획** 충실히 준비
4. 2등급 기기 **적합성 평가** 시 기술 문서 심사 + 제조 및 품질관리 심사 모두 필요
