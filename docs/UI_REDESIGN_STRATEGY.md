# HnVUE UI Redesign Strategy

## 현재 전략

HnVUE UI 개정은 더 이상 HTML/Pencil/Figma 목업 우선 방식으로 진행하지 않는다. 활성 전략은 `PPT 최종안 -> PPT 분석 -> UISPEC -> Coordinator handoff` 순서다.

## 기준 원칙

- 기준 원천은 `docs/★HnVUE UI 변경 최종안_251118.pptx`
- `변경 2안`이 있는 슬라이드는 `2안`을 최종안으로 채택
- 대형 화면은 독립 창으로 해석
- 현재 XAML은 설계 진실이 아니라 구현 참조물
