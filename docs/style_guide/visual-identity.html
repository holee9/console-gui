Spacing System

The spacing system is based on a 4px base unit with consistent scale.

Spacing Scale

xs: 4px
Micro spacing, tight gaps
sm: 8px
Small spacing, icon padding
md: 16px
Default spacing, component gaps
lg: 24px
Large spacing, section margins
xl: 32px
Extra large, component groups
2xl: 48px
Section spacing
3xl: 64px
Page-level spacing

XAML Resource Usage

Using Colors

XAML 
<!-- Reference from Theme Resources -->
<TextBlock Text="Primary Text"
           Foreground="{StaticResource Primary.Main}"/>

<Border Background="{StaticResource Neutral.Surface}"
        BorderBrush="{StaticResource Neutral.Border}"
        BorderThickness="1"/>

Using Typography

XAML 
<!-- Apply predefined text styles -->
<TextBlock Text="Screen Title"
           Style="{StaticResource TextStyle.H1}"/>

<TextBlock Text="Body content goes here"
           Style="{StaticResource TextStyle.Body}"/>

<!-- Manual typography -->
<TextBlock Text="Custom text"
           FontFamily="{StaticResource Font.Primary}"
           FontSize="{StaticResource Size.Body}"
           FontWeight="{StaticResource Weight.Regular}"/>

Using Spacing

XAML 
<!-- Standard spacing values -->
<Grid Margin="{StaticResource Spacing.Md}">
    <!-- Content -->
</Grid>

<StackPanel Spacing="{StaticResource Spacing.Sm}">
    <!-- Items -->
</StackPanel>

Accessibility Guidelines

Color Contrast

All color combinations meet WCAG 2.2 Level AA requirements:

  • Normal text: Minimum 4.5:1 contrast ratio
  • Large text (18px+): Minimum 3:1 contrast ratio
  • UI components: Minimum 3:1 contrast ratio

Color Independence

Do

Use color alongside text labels and icons

Don't

Rely on color alone to convey meaning
