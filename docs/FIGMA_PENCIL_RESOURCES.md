## 의료기기 UI 디자인 원칙

### 색상 (Color)
- Primary: #1B4F8A (MahApps.Metro Blue)
- Success: #2ED573
- Warning: #FFA502
- Error: #FF4757
- Background: #1A1A2E (Dark Mode)

### 타이포그래피 (Typography)
- Primary: Segoe UI / Malgun Gothic (한국어)
- Display: 32px Bold
- H1: 24px Semibold
- Body: 14px Regular

### 간격 (Spacing)
- Base Unit: 4px
- XS: 4px, SM: 8px, MD: 16px, LG: 24px, XL: 32px

### 컴포넌트 (Components)
- Button: 36/44px height, 6px radius
- Input: 36px height, 8px padding
- Card: 12px radius, 16px padding
