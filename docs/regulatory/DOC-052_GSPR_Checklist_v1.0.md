## 1. 문서 목적 및 범위

### 1.1 문서 목적

본 문서는 EU MDR 2017/745 Annex I에 규정된 **일반 안전성 및 성능 요건 (General Safety and Performance Requirements, GSPR)** 전체 23개 조항에 대해 HnVue Console SW (HnVue)의 적용 여부, 준수 증거 표준, 증빙 문서(DOC-XXX), 준수 상태를 체계적으로 매핑하여 Notified Body(NB) 기술 문서 심사의 기준 문서로 제공한다.

본 문서는 DOC-037(CE 기술 문서)에서 부분 커버하던 8개 GSPR 조항을 포함하여, Annex I 전체 23개 조항에 대한 완전한 독립 GSPR 체크리스트 문서로서 기능한다.

### 1.2 대상 제품

| 항목 | 내용 |
|------|------|
| 제품명 | HnVue Console SW (HnVue) |
| 제품 유형 | 의료기기 소프트웨어 (SaMD — Software as a Medical Device) |
| SW 안전 등급 | IEC 62304 Class B |
| 의도된 용도 | 디지털 방사선 촬영(DR) 워크스테이션 제어 및 영상 취득·처리·전송 |
| 배포 환경 | Windows 10/11 IoT Enterprise LTSC (병원 내부 폐쇄망) |
| SW 버전 | v1.0 (개발 예정) |
| 연결 인터페이스 | DICOM (PACS/RIS), HL7 (HIS), X-Ray Generator Control, LDAP/AD |

## 5. N/A 처리 사유 요약

HnVue Console SW는 **SW 전용 의료기기(SaMD — Software as a Medical Device)**로서, 하드웨어 없이 소프트웨어만으로 구성된 특성상 다음 GSPR 조항은 해당 없음(N/A)으로 처리한다.

| GSPR § | N/A 처리 사유 |
|--------|-------------|
| §7 (이식형 기기 특별 요건) | HnVue Console SW는 이식형(Implantable) 기기가 아님. 환자 신체에 이식·삽입되는 구성요소 없음 |
| §10, §10.1, §10.2, §10.3 (화학적·생물학적 특성) | SW 전용 제품으로 물리적 재료 구성 없음. 환자 신체 접촉 없음. 화학적·생물학적 위험 경로 없음 |
| §11 (감염·미생물 오염 방지) | SW 전용 제품으로 미생물 오염 경로 없음 |
| §12 (포장 및 멸균) | 전자 배포(Electronic Distribution) 제품. 물리적 포장·멸균 불필요. SW 배포 무결성은 코드 서명으로 보장 (DOC-046 §3 통제 4) |
| §15 (방사선 보호 요건) | SW는 방사선을 발생·조사하지 않음. DR 검출기 및 X-Ray Generator는 HnVue Console SW와 별도 허가를 받는 하드웨어 의료기기임 |
| §17 (능동 이식형 기기 전기적 요건) | 이식형 기기 아님 |
| §18 (능동 이식형 기기 전자기적 요건) | 이식형 기기 아님 |
| §19 (EMC/전자기 교란) | SW 전용 의료기기. 전자기기 하드웨어 구성요소 없음. HnVue SW가 설치되는 PC 워크스테이션 자체는 CE 마킹이 완료된 상용 제품으로, EMC 요건이 PC 제조사에 의해 충족됨. HnVue SW 설치로 인해 EMC 특성이 변경되지 않음 |
| §22 (이식형 기기 특수 요건) | 이식형 기기 아님 |
