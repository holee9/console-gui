## 측정 방법

### SUS 점수 측정
1. 설문지 10문항 배포 (방사선사, 의사, 관리자 각 그룹)
2. 각 문항 0-4점 응답
3. 공식: `(홀수 문항 합 - 5 + 25 - 짝수 문항 합) × 2.5`
4. 참여자 최소 5명, 권장 15명

### 과업 완료 시간 측정
1. 스크린 레코딩 소프트웨어로 세션 녹화
2. 각 과업 시작/종료 타임스탬프 기록
3. 5회 반복 측정, 평균값 기록
4. 기준선 대비 변화율 계산

### 휴리스틱 평가
1. Nielsen 10 원칙 + 의료기기 기준 (CHANGE_MANAGEMENT_PROCESS.md §Step 2)
2. 평가자 최소 3명 (독립 평가 후 합산)
3. 화면별 0-10점 채점 → 가중 합산
