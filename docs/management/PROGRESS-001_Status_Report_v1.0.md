### 4.2 Gantt 읽기 포인트

1. **현재 위치(▼) 기준 왼쪽**: 이미 소진된 MM이 어디에 투입되었는지
2. **현재 위치(▼) 기준 오른쪽**: 잔여 MM이 어디에 필요한지
3. **!!!! (CRIT)**: 계획상 아직 정시이나, 하드웨어 의존으로 착수 차단 우려
4. **▓▓▓▓ (진행중)**: 계획 대비 선행 착수한 항목 (AI 병행 효과)
5. **████ (완료)**: 2.3 MM으로 이미 완료된 항목
